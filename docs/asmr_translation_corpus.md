# ASMR 日译中语料库与书写规范

用途：为后续 ASMR 烤肉提供可持续更新的术语、错词、语气、场景表达参考。每次完成一个作品后都要更新本文件。

当前版本：2026-06-12  
首次来源作品：`RJ01201653`，有初稿台本，含本篇、EX、促销/试听音频。

## 使用规则

- 翻译前先读本文件。
- 翻译时把本文件作为参考，不要机械套用；不同作品的人设、关系、时代、方言会改变译法。
- 本文件记录的模型和工具只代表当次项目经验，不是后续项目的硬性要求。
- 校对后把新经验追加到对应小节。
- 不确定的内容写入“待验证”，不要写成定论。
- 每条新增内容尽量包含来源作品、轨道、上下文和日期。

## 新增条目格式

建议使用以下格式：

```markdown
### 术语或问题名

- 来源：RJxxxx / 轨道名或编号 / 有无台本 / 日期
- 日文或 ASR：...
- 推荐译法：...
- 不推荐译法：...
- 说明：为什么这样译，适用条件是什么
- 标签：术语 / ASR错词 / 语气 / 拟声 / 人称 / 成人场景 / 待验证
```

## 总体中文风格

- 目标是“能挂字幕、读得顺、符合情境”，不是论文式逐字翻译。
- 亲密、撒娇、挑逗、犹豫、停顿要保留，但中文要自然。
- 对角色语气要统一：同一角色的敬语、调侃、害羞、强势程度不能前后乱跳。
- 同一句里不要混太多英文、日文和中文；必要术语如 `coser`、`cos`、`M 男` 可以保留。
- 长段重复音效不要逐字铺满字幕，可简化为短拟声或方括号提示。
- 不要把 ASR 幻觉翻得很认真；明显荒唐时要回看日文、台本或上下文。
- 有台本项目以台本为主要参考。模型质检只做辅助补漏；当模型建议与台本、相邻剧情或角色设定冲突时，优先相信台本和上下文。
- 初翻完成后必须做模型质检。尤其是无台本项目，很多问题不是孤立错词，而是 ASR 同音误识别带来的整段逻辑错位；单靠关键词扫描和人工粗读不够稳。
- 模型质检的结果只能作为候选。批量套用建议后必须回看相邻字幕并再次扫雷，因为 QC 模型也可能把可接受句子改成新的生硬或错误表达。

## SRT 书写规范

- 不改时间轴，不改编号，除非用户明确要求重切时间轴。
- 每条字幕尽量简短，适合屏幕阅读。
- 一条里可用换行，但不要让句子太长。
- 校对拟声词时要对照日文台本或日文 ASR，先判断音效类型，再压缩成中文提示。不要只按中文重复字符猜成 `[亲吻声]`、`[摩擦声]` 等。
- 拟声词可简化：
  - `[亲吻声]`
  - `[舔耳声]`
  - `[亲吻声和舔耳声]`
  - `[长长的喘息声、亲吻声、舔耳声]`
  - `撸撸……`
  - `啪啪、啪啪……`
- 长段无意义 ASR 幻觉，如“感谢观看/请订阅”，如果实际是音效，改成对应音效提示。

## 常见语气处理

### 女主对“先生”的称呼

- 来源：`RJ01201653` / 全篇 / 有台本 / 2026-06-12
- 推荐译法：保留 `老师`
- 说明：不要自动改成“你”或“先生”。可根据句子自然度混用“老师/你”，但角色称呼要稳定。
- 标签：人称 / 语气

### 撒娇式命令

- 来源：`RJ01201653` / 02、07、促销03、10 / 有台本 / 2026-06-12
- 推荐译法：`来，快说`、`要好好配合我的声音哦`、`别一下子就不行哦`
- 说明：既有命令感，也要保留亲密关系，不要翻成冷冰冰命令句。
- 标签：语气

### 羞耻台词复述

- 来源：`RJ01201653` / 02、促销03、10 / 有台本 / 2026-06-12
- 规则：女主让男方说的话要加引号或明确人称。
- 推荐译法：`“求求你，帮我撸撸小弟弟嘛”`
- 常见错误：把男方要复述的话译成女主自己在说。
- 标签：人称 / 易错

### 喘息和停顿

- 来源：`RJ01201653` / 全篇 / 有台本 / 2026-06-12
- 推荐译法：`哈啊……`、`嗯……`、`呼……`
- 规则：保留少量情绪，不要每个喘息都铺满屏幕。
- 标签：拟声 / 语气

## ASR 高风险错词清单

翻译后必须搜索并人工确认：

- `感谢观看`、`感谢您的观看`：常是长段音效或结束噪声幻觉。
- `订阅`、`字幕`、`语言模型`、`JSON`：通常是模型或 ASR 幻觉。
- `冥想神经`：多半应为 `迷走神经`。
- `正义`：成人语境中常应为 `精液`。
- `写生`：可能是 `射精`，需看上下文。
- `写真`、`书生`、`照片`：成人 ASMR 中可能来自 `射精` 的 ASR 误识别，需结合倒数、高潮、射精训练等上下文确认。
- `魔女`：可能是 `M男`，需看上下文。
- `手呼吸`：常是 `手コキ` 或手冲动作。
- `尿尿`、`嘘嘘`：在 `シコ/しっこ` 语境下常应为 `撸撸`。
- `僧侣`、`骚动`、`Souro`、`灵魂`、`步态`、`神父`：在早漏主题作品中常是 `早漏（そうろう）`。
- `主人/服侍主人/オーナー`：可能是 `オナニー` 被误识别，需结合自慰训练语境修正。
- `口交`：若源文是 `オナニー/ホナニー`，多半应为“自慰/手淫”，不要误套 `フェラ`。
- `道谢室`、`草子`：可能是 `童貞喪失`。
- `警察`：可能是相近音误识别，必须回看日文。
- `大开杀戒`：可能是 `し放題`，应按“想怎么做都可以”理解。
- `生涩`：可能是 `生エッチ`，应为“无套做爱”。
- `耳舐め`：中文成品里应改成“舔耳”。
- `Love Hotel（爱情旅馆）`：改成“情人旅馆”。
- `感谢观看/感谢您的观看`：无台本 ASR 结尾常见幻觉。若上下文只是尾部静音或音效，可改为 `[结尾音效]`，不要保留成剧情台词。

### RJ01533156

- 日期：2026-06-12
- 类型：无台本；本篇与短轨均生成独立 SRT
- 本次使用过的工具组合：`mlx-community/whisper-large-v3-mlx-4bit` + oMLX `Qwen3.6-27B-MLX-VL-oQ6`。
- 关键经验：

  - QC 建议不可照单全收。本次自动建议中也出现过生硬替换，需要人工回看相邻字幕后再决定是否应用。
