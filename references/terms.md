# ASMR 日译中确认术语

翻译前和 QC 修正前读取。不同作品的人设、关系、时代、方言会改变译法；不要机械套用。

## レイヤー / コスプレイヤー

- 来源：`RJ01201653` / 全篇 / 有台本 / 2026-06-12
- 推荐译法：`coser`、`角色扮演者`，口语场景优先 `coser`
- 说明：DLsite 成人向 ASMR 中 `レイヤー` 常指 coser，不是“图层”。
- 标签：术语

## 公式レイヤー

- 来源：`RJ01201653` / 01、02、促销02 / 有台本 / 2026-06-12
- 推荐译法：`官方 coser`
- 不推荐译法：`官方解说员`、`官方 Coser` 大小写混乱
- 说明：指受官方邀请/认证的 coser。
- 标签：术语 / ASR错词

## 売り子

- 来源：`RJ01201653` / 01、促销02 / 有台本 / 2026-06-12
- 推荐译法：`看摊`、`帮你卖东西`、`摊位店员`
- 说明：同人展语境中指摊位帮卖的人，不宜机械译为“售货员”。
- 标签：术语 / 场景

## コスエッチ / コスプレエッチ

- 来源：`RJ01201653` / 01、促销02 / 有台本 / 2026-06-12
- 推荐译法：`cosplay 做爱`、`穿着 cos 服做爱`
- 可按语境简化：`cos 做爱`
- 不推荐译法：`Cosplay 色情`、`cosplay 情境`
- 标签：术语 / 成人场景

## ドレスコス / 着物コス

- 来源：`RJ01201653` / 05、06、07、促销06、10 / 有台本 / 2026-06-12
- 推荐译法：`礼服 cos`、`和服 cos`、`穿着礼服/和服 cos`
- 说明：如果直接说“服装”会丢掉 cosplay 设定，可保留 `cos`。
- 标签：术语

## 手コキ / シコシコ / しっこ

- 来源：`RJ01201653` / 02、07、促销03、04、10 / 有台本 / 2026-06-12
- 推荐译法：`手冲`、`用手帮你弄`、`撸`、`撸撸`
- 不推荐译法：`手呼吸`、`尿尿`、`嘘嘘`、`揉小弟弟`
- 说明：ASR 很容易把 `シコ/しっこ` 识别成接近排尿或无意义词。成人 ASMR 语境下通常是手冲拟声。
- 标签：术语 / ASR错词 / 拟声

## 早漏 / そうろう

- 来源：`RJ01533156` / 全篇 / 无台本 / 2026-06-12
- 日文或 ASR：`僧侶`、`騒動`、`ソウロウ`、`Souro`
- 推荐译法：`早漏`、`早漏先生`、`早漏小弟弟`
- 不推荐译法：`僧侣`、`骚动`、`灵魂`、`步态`、`神父`、`Souro`
- 说明：本作核心设定是“早漏改善指导”。ASR 极易把 `そうろう` 错识别成同音/近音词，翻译时必须结合标题和上下文统一修正。
- 标签：术语 / ASR错词 / 成人场景 / 无台本验证

## ホナニー / オナニー

- 来源：`RJ01533156` / 01、02 / 无台本 / 2026-06-12
- 日文或 ASR：`ホナニー`、`オーナーに`
- 推荐译法：`自慰`、`手淫`、按语气可译为 `撸`
- 不推荐译法：`口交`
- 说明：ASR 可能把 `オナニー` 识别成变体假名串。没有 `フェラ` 语境时不要译成口交。
- 标签：术语 / ASR错词 / 成人场景 / 无台本验证

## オナサポ

- 标签：术语 / 成人场景
